# 🏢 智享HR - 企业人力资源管理平台（精修版）

## 📋 项目概述

**平台名称**：智享HR Cloud  
**定位**：面向中大型企业的一体化人才管理平台  
**核心价值**：AI驱动的智能化人才全生命周期管理  
**对标产品**：北森iTalentX

## 🔹 模块一：组织与员工管理

### 1.1 组织架构管理

**精细功能**：
| 功能点 | 说明 |
|--------|------|
| 多层级组织树 | 支持集团-子公司-部门-团队四级架构 |
| 岗位体系 | 岗位族/岗位序列/岗位等级三层体系 |
| 编制管理 | 部门编制规划、超编预警、编制调整流程 |
| 组织变更 | 部门拆分/合并/撤销，历史版本追溯 |
| 汇报关系 | 实线汇报+虚线汇报，矩阵式管理支持 |

**岗位体系示例**：
```
技术族
├── 研发序列
│   ├── 初级工程师（P1-P2）
│   ├── 中级工程师（P3-P4）
│   ├── 高级工程师（P5-P6）
│   └── 技术专家（P7-P8）
└── 产品序列
    ├── 产品专员（P1-P2）
    ├── 产品经理（P3-P4）
    └── 高级产品经理（P5-P6）
```

### 1.2 员工信息管理

**员工画像**（参考北森人才画像）：
| 维度 | 内容 |
|------|------|
| 基本信息 | 姓名、性别、年龄、学历、专业 |
| 工作信息 | 部门、岗位、职级、入职日期、司龄 |
| 能力标签 | 技能、证书、项目经验 |
| 绩效表现 | 历史绩效等级、潜力评级 |
| 发展轨迹 | 岗位变动、培训经历、晋升记录 |

**员工生命周期事件**：
- 入职、转正、调岗、晋升、降职、离职
- 每个事件自动触发相关流程和通知

### 1.3 人才盘点（核心亮点）

**人才九宫格**：
```
        绩效表现
     高    中    低
    ┌────┬────┬────┐
高  │A类 │B类 │C类 │
潜  │明星 │骨干 │待观察│
力  ├────┼────┼────┤
中  │D类 │E类 │F类 │
    │潜力 │稳定 │需提升│
    ├────┼────┼────┤
低  │G类 │H类 │I类 │
    │待发展│普通 │淘汰 │
    └────┴────┴────┘
```

**盘点流程**：
```
设定盘点周期 → 收集数据（绩效+能力+潜力）→ 人才评审会 → 生成九宫格 → 制定人才策略
```

**人才策略矩阵**：
| 人才类型 | 策略 |
|----------|------|
| A类（明星） | 重点培养、快速晋升、股权激励 |
| B类（骨干） | 持续激励、技能提升、职业规划 |
| C类（待观察） | 针对性辅导、设定改进目标 |
| I类（淘汰） | 转岗或解除劳动关系 |

### 1.4 合同与档案管理

**合同管理**：
| 功能 | 说明 |
|------|------|
| 合同模板 | 劳动合同、保密协议、竞业限制 |
| 合同签订 | 电子合同、签字流程 |
| 到期提醒 | 合同到期前30/60/90天提醒 |
| 续签管理 | 续签审批流程 |

**电子档案**：
- 档案分类：基本信息、合同文件、薪酬记录、绩效记录、培训记录
- 版本管理：记录每次修改历史
- 权限控制：按角色分级查看

## 🔹 模块二：招聘管理

### 2.1 招聘需求管理

**需求提报流程**：
```
用人部门提需求 → HR审核 → 部门负责人审批 → 需求确认
```

**需求表单字段**：
| 字段 | 说明 |
|------|------|
| 岗位名称 | 如：高级Java开发工程师 |
| 所属部门/团队 | 明确归属 |
| 招聘人数 | 计划招聘数量 |
| 到岗时间 | 期望入职时间 |
| 岗位级别 | 对应岗位体系 |
| 薪酬范围 | 薪资区间 |
| 关键任职要求 | 核心技能、经验要求 |

### 2.2 AI智能招聘（核心亮点）

**AI简历筛选**：
| 功能 | 说明 |
|------|------|
| 智能解析 | 自动提取简历关键信息 |
| 技能匹配 | 与岗位要求智能匹配打分 |
| 相似度分析 | 分析候选人与岗位的匹配度 |
| 人才推荐 | 基于历史数据推荐合适候选人 |

**AI面试助手**：
- 面试问题智能生成
- 面试评价自动记录
- 候选人能力画像生成

### 2.3 候选人管理

**候选人才库**：
| 分类 | 说明 |
|------|------|
| 活跃候选人 | 当前正在招聘流程中的 |
| 储备人才 | 未录用但潜力不错的 |
| 黑名单 | 不符合公司要求的 |

**候选人体验优化**：
- 申请进度实时查询
- 面试反馈及时通知
- 个性化邮件/短信通知

### 2.4 面试流程管理

**面试流程定制**：
```
简历初筛 → 线上测评 → HR初面 → 业务面试 → 高管终面 → Offer
```

**面试评估表**：
| 维度 | 评分标准 | 权重 |
|------|----------|------|
| 专业能力 | 专业知识、技能水平 | 30% |
| 工作经验 | 相关经验、项目经历 | 25% |
| 综合素质 | 沟通能力、团队协作 | 25% |
| 文化匹配 | 价值观契合度 | 20% |

### 2.5 Offer与入职管理

**Offer流程**：
```
Offer审批 → 发送候选人 → 候选人接受 → 背景调查 → 入职准备
```

**入职准备清单**：
- IT设备准备（电脑、账号）
- 工位安排
- 入职引导人安排
- 培训计划制定

## 🔹 模块三：绩效管理

### 3.1 目标管理（OKR/KPI双模式）

**OKR模式**：
| 要素 | 说明 |
|------|------|
| Objective | 目标（定性） |
| Key Results | 关键结果（3-5个，定量） |
| Initiatives | 行动计划 |

**KPI模式**：
| 要素 | 说明 |
|------|------|
| 指标名称 | 如：销售额、客户满意度 |
| 目标值 | 期望达成的数值 |
| 权重 | 该指标占比 |
| 计算方式 | 实际值/目标值×100% |

**目标对齐**：
```
公司OKR → 部门OKR → 团队OKR → 个人OKR
    ↑         ↑         ↑         ↑
   对齐      对齐      对齐      对齐
```

### 3.2 持续绩效管理（核心亮点）

**绩效日历**：
- 周/月度目标跟踪
- 定期1:1面谈记录
- 即时反馈收集

**360度评估**：
| 评估维度 | 说明 |
|----------|------|
| 上级评估 | 直属主管评价 |
| 下级评估 | 下属评价（针对管理者） |
| 同事评估 | 跨部门协作评价 |
| 自我评估 | 员工自评 |

### 3.3 绩效评估流程

**评估周期**：
| 周期 | 适用场景 |
|------|----------|
| 季度评估 | 研发、市场等快速迭代团队 |
| 半年度评估 | 职能部门 |
| 年度评估 | 全员综合评估 |

**评估流程**：
```
目标回顾 → 自评 → 上级评估 → 360评估 → 绩效校准会 → 结果沟通 → 申诉处理
```

**绩效校准**：
- 确保不同部门评估标准一致
- 控制各等级比例分布

### 3.4 结果应用

**绩效与薪酬联动**：
| 绩效等级 | 调薪幅度 | 年终奖金系数 |
|----------|----------|--------------|
| S | 15-20% | 2.0-2.5 |
| A | 10-15% | 1.5-2.0 |
| B | 5-10% | 1.0-1.5 |
| C | 0-5% | 0.5-1.0 |
| D | 0% | 0 |

**发展建议**：
- 根据评估结果制定个人发展计划
- 推荐针对性培训课程

## 🔹 模块五：薪酬管理

### 5.1 薪酬体系设计

**薪酬结构**：
```
Total Compensation = 固定薪酬 + 浮动薪酬 + 长期激励 + 福利
```

| 组成部分 | 说明 |
|----------|------|
| 固定薪酬 | 基本工资 + 津贴补贴 |
| 浮动薪酬 | 绩效奖金 + 项目奖金 + 年终奖金 |
| 长期激励 | 股权激励 + 限制性股票 |
| 福利 | 法定福利 + 补充福利 |

**职级薪酬表**：
| 职级 | 基本工资范围 | 绩效奖金比例 |
|------|--------------|--------------|
| P1 | 6k-8k | 10% |
| P2 | 8k-10k | 15% |
| P3 | 10k-15k | 20% |
| P4 | 15k-20k | 25% |
| P5 | 20k-30k | 30% |

### 5.2 薪资核算

**核算流程**：
```
数据采集 → 计算处理 → 审核确认 → 发放执行 → 报表生成
```

**自动化核算**：
- 自动读取考勤数据
- 自动计算绩效奖金
- 自动扣除社保公积金
- 自动计算个人所得税

**异常处理**：
- 薪资异常自动检测
- 异常原因分析
- 人工复核流程

### 5.3 福利管理

**福利体系**：
| 类别 | 项目 |
|------|------|
| 法定福利 | 五险一金 |
| 补充福利 | 补充医疗、商业保险、年度体检 |
| 弹性福利 | 餐补、交通补贴、通讯补贴、节日福利 |
| 长期激励 | 股权激励计划 |

**福利商城**：
- 员工可自主选择福利项目
- 积分兑换福利商品
- 福利使用记录查询

### 5.4 薪酬分析

**分析维度**：
| 维度 | 说明 |
|------|------|
| 人力成本分析 | 部门/公司人力成本趋势 |
| 薪酬结构分析 | 固定/浮动/福利占比 |
| 薪酬竞争力 | 与市场对比 |
| 薪酬公平性 | 内部公平性分析 |
